    • LeanCloud + Weibo + Web 实现第三方登录

    本篇文章主要介绍了使用 leancloud + weibo 的方法在web页面上实现登录功能

    前期准备

    • 注册 leancloud 账号
    • 注册 微博开放平台 账号

    登录流程

    1. 打开登录页面，点击登录按钮进行跳转
    2. 跳转按钮会跳转到微博登录的授权页面，授权成功后再跳转回应用主页
    3. 回到主页后通过微博返回的code参数，调用后台API请求用户的access_token
    4. 根据请求回的用户 access_token、expires_in、uid,调用leancloud的API进行
    5. 登录成功后即可使用getCurrentUser()方法获取到现有的用户信息

    详细流程

    注册微博开发者账号，获取 App Key 和 App Secret

    登录微博发放平台进行注册后，进入我的应用找到你创建的web应用。这里面有几个需要注意的地方：

    • App Key
    • App Secret
    • 授权回调页面

    其中前两者是用来调用微博的api所需要的参数,回调页面是指当用户点击授权后跳转到的页面.这里设置为你的主页即可.

    创建登录页面

    创建一个最简单的登录跳转链接,其中：

    • client_id: 这里填写的是上一步创建应用时获取到的 App Key
    • redirect_uri: 用户认证完后你希望的跳转页面,由于是在本地测试，直接填写本地的应用地址即可

    用户认证完后回跳转到 http://127.0.0.1:7001?code=********, 其中的code是我们下一步用来获取用户的access_token时使用的

    获取access_token

    • client_id: 应用的 App Key
    • client_secret: 应用的 App Secret
    • code: 上一步我们获取到的 code
